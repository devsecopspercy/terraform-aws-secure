# terraform-aws-secure

Define una infraestructura segura en AWS con Terraform, incluyendo:
✅ VPC privada (sin acceso público por defecto).
✅ Subred privada sin IPs públicas.
✅ Security Group con SSH restringido a una IP específica y HTTPS global.
✅ Instancia EC2 sin IP pública.
✅ Disco duro cifrado para mayor seguridad.
